How It Works

The bottom player uses a single WaveformPlayer instance with showControls: false and showInfo: false for a waveform-only display. When you click a track, it calls loadTrack() to seamlessly switch audio without destroying the player.

External controls (play/pause, prev/next) call the player's API methods like togglePlay(), while the waveform remains fully interactive for seeking. The queue is managed in plain JavaScript — tracks are added as you click them, and you can reorder your listening from the queue panel.

This pattern works great for music apps, podcast platforms, beat stores, or any site where you want uninterrupted playback while users browse content.
